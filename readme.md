# Crime Intelligence System 🕵️

A CLI-based crime intelligence database built in Python.

## Features
- Log crimes with full details
- Add criminal profiles
- Search crimes by location, type, status
- Search criminals by name, danger level
- Pattern analysis using Counter and defaultdict
- Cold case detection using datetime
- Stats dashboard
- Generate investigation reports as .txt files

## Topics Covered
- Dictionaries, Lists, Sets, Tuples
- File handling (JSON + TXT)
- datetime module
- collections (Counter, defaultdict, deque, namedtuple)
- Error handling

## How to Run
python MINI_PROJECT_INVESTIGATOR.py
